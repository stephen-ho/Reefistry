# Reefistry

<img src="https://github.com/stephen-ho/Reefistry/blob/main/dashboard.png" width="250"/>

Project Description
----------------
This app is a simple-to-use logging software to keep track of your aquarium's water parameters. The dashboard renders your most recent test data with color and arrows compared with previously entered data and acceptable ranges based on natural seawater. Click into each parameter to see a more in-depth log of your past 5 data points and a line graph to track stability.

Future Functionality:
- add/remove aquariums
- add/remove parameters
- control how many data points show in the in-depth page
- overlay graphs to track correlation between multiple parameters


Dependencies
----------------
- MongoDB
- Express
- Axios
- React
- Node.js

Instructions to set up repository
----------------

1. Fork and clone this repository locally
2. Navigate to the local repository folder
3. Install dependencies (npm install)
3. Launch dev environment (npm run server)
4. Launch client environment (npm run start)
5. Navigate to http://localhost:3000/
